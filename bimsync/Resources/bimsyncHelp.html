General Usage Instructions

Connect to your bimsync® account.

  • Click on Login
  • A browser will open, prompting for your bimsync® credentials
  • Click on "Allow access" to allow the plugin to access your bimsync® account

Upload your model to bimsync®

  • Click on Upload
  • Select the bimsync® project and the bimsync® model and where to upload your Autodesk® Revit® file
  • Select an existing IFC Export configuration or use the bimsync® default
  • Type a comment to describe your upload
  • Click on Upload

The plugin will export your model as an IFC file, then upload it to the selected bimsync® model. This model will be saved in your Revit file to be remembered for the next upload.

Setup your IFC Export settings

The plugin use of the open source IFC plugin for Autodesk® Revit®. It is recommended to use the last version, available here.

The default export setup (<bimsync Setup>) will work fine with bimsync®. However, if you want another export setup, you can create a new one by using the Revit® IFC Export interface. This new setup will be available in on Upload.

Installation/Uninstallation

The installer that ran when you downloaded this plug-in from Autodesk App store has already installed the plug-in. You may need to restart the Autodesk product to activate the plug-in. To uninstall this plug-in, exit the Autodesk product if you are currently running it, simply rerun the installer by downloading it again from Autodesk App store, and select the 'Uninstall' button, or you can uninstall it from 'Control Panel\Programs\Programs and Features' (Windows 7/8/8.1/10), just as you would uninstall any other application from your system.
